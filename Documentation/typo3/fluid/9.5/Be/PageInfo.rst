===========
be.pageInfo
===========


ViewHelper which return page info icon as known from TYPO3 backend modules.

.. note::
   This ViewHelper is experimental!

Examples
========

Default::

   <f:be.pageInfo />

Page info icon with context menu

Arguments
=========


This ViewHelper has no arguments.
